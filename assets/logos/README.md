# KeLiva Logo Assets

This directory contains all logo variants for the KeLiva project.

## Available Variants

### Primary Logo
- `logo-full.png` - Full logo with tagline (1024x1024px) - Use for presentations, marketing materials

### Recommended Additional Variants (To be created)
- `logo-icon.png` - Icon only (512x512px) - Use for app icons, favicons
- `logo-horizontal.png` - Horizontal layout without tagline - Use for headers, navigation bars
- `logo-dark.png` - Version optimized for dark backgrounds
- `logo-light.png` - Version optimized for light backgrounds

## Usage Guidelines

### Web Dashboard (Study Room)
- Use `logo-horizontal.png` in the navigation bar (max height: 40px)
- Use `logo-icon.png` for favicon (32x32px, 16x16px)

### WhatsApp Integration
- Profile picture: `logo-icon.png` (cropped to circular)

### Mobile App (Future)
- App icon: `logo-icon.png` with appropriate padding for iOS/Android

### Documentation
- Use `logo-full.png` for README headers and documentation

## Color Palette

- Primary Navy: `#2D2D44` (or similar)
- Accent Orange: `#FF8C42` (or similar)
- Background Cream: `#F5F1E8` (or similar)

## Notes

- All logos should maintain the 1:1 aspect ratio for the icon
- Ensure minimum clear space around logo equals the height of the sound wave element
- Never distort or skew the logo
- Maintain color consistency across all variants
